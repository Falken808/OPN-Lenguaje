Inicio rapido

OPN en 5 minutos

Crea tus primeros programas, usa variables y funciones, y ejecuta desde Python o desde opn.exe en Windows.

Paso 1: primer archivo

hola.opn

print("Hola, OPN!");

Ejecucion

python opn2.py hola.opn
.\opn.exe hola.opn

Paso 2: variables

var edad = 25;
var nombre = "Carlos";
var altura = 1.75;

print("Nombre: " + nombre);
print("Edad: " + edad);
print("Altura: " + altura + " metros");

Paso 3: funciones

function suma(a, b) {
    return a + b;
}

print(suma(10, 20));
Para produccion: python opn2.py compile app.opn -o app.py
