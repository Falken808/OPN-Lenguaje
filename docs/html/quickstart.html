Quickstart

Get running in under 10 minutes

Create your first file, run it, compile it, install dependencies, and build a portable binary.

Step 1: create a file

hello.opn

var greeting = "Hello BluePanda";
print(greeting);

Step 2: run it

opn hello.opn

Step 3: compile to Python

opn compile hello.opn -o hello.py
python hello.py

Step 4: dependencies and build

opn -m pip install pygame
opn build hello.opn -o dist/hello
