## <img src="https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/windows.svg" width="22" /> Instalación en Windows

Instalar OPN es directo y sin configuraciones adicionales.

1. Descarga el paquete de **OPN Script**.  
2. Ejecuta `instalar.bat`.  
3. El comando `opn` quedará disponible en tu terminal.

```bash
# Verifica tu instalación
opn --version
```

## <img src="https://raw.githubusercontent.com/simple-icons/simple-icons/develop/icons/flash.svg" width="22" /> Inicio Rápido

Ejecutar código en OPN es inmediato:

```javascript
// archivo: hola.opn
var saludo = "Hola Mundo";
print(saludo);

function sumar(a, b) {
    return a + b;
}
```

### Comandos básicos

| Acción | Comando |
|-------|---------|
| Ejecutar | `opn programa.opn` |
| Compilar a Python | `opn compile programa.opn -o salida.py` |
| Modo Watch | `opn watch programa.opn` |
